1 Byte and The number of bits(1ByteВЎђ bitВѕў)

1bit(в╣ёьіИ)віћ 0Ж│╝ 1ВЮў Ж░њВЮё Ж░ђВДѕ Вѕў Въѕвіћ вЇ░ВЮ┤ьё░ВЮў ВхюВєї вІеВюёВъЁвІѕвІц.

ВЮ┤ bitвЦ╝ ьі╣ВаЋЖ░»Вѕў вДїьЂ╝ вгХВќ┤Вёю, 1byte(в░ћВЮ┤ьіИ)вЮ╝Ж│а ьЋЕвІѕвІц.

ByteВЮў ВЮўв»И

ВхюВ┤ѕВЮў byteвіћ РђўьЋўвѓўВЮў character вЇ░ВЮ┤ьё░вЦ╝ вѓўьЃђвѓ┤віћвЇ░ ьЋёВџћьЋю bitВѕўРђЎвЦ╝ ВЮўв»ИьќѕВіхвІѕвІц.

Byte┬аdenotes a group of bits used to encode a character, or the number of bits transmitted in parallel to and from input-output units.
- from РђўPLANNING A COMPUTER SYSTEMРђЎ

В╗┤ьЊеьё░ВЌљВёа, characterвЦ╝ ьі╣ВаЋ ВаЋВѕў(bitвАю ьЉюьўёьЋю integer)вАю mappingьЋўВЌг ВѓгВџЕьЋЕвІѕвІц.

В┤ѕЖИ░ В╗┤ьЊеьё░віћ ВБ╝вАю textвЇ░ВЮ┤ьё░вЦ╝ В▓ўвдгьќѕЖИ░ вЋївгИВЌљ, ВъљВЌ░ВіцвЪйЖ▓ї byteЖ░ђ вЕћвфевдг Ж│хЖ░ёВЮё ьЋавІ╣ьЋўвіћ ВхюВєї вІеВюёЖ░ђ вљўВЌѕВіхвІѕвІц(characterвЦ╝ ВађВъЦьЋ┤ВЋ╝ьЋўЖИ░ вЋївгИВЌљ). ВЮ┤ьЏё ВѕўвДјВЮђ Рђўcomputer architectureРђЎВЌљВёю вЕћвфевдгВЮў Ж│хЖ░ёВЮё вѓўьЃђвѓ┤віћ ВхюВєї вІеВюёвАю ВѓгВџЕвљўВЌѕВіхвІѕвІц.

ьўёВъгВ▓ўвЪ╝ 1byte = 8bitВЮИ ьЉюВцђВЮ┤ ВЌєВЮёвЋївіћ, ByteВЮў bitВѕўвіћ HardwareВЌљ вћ░вЮ╝ вІгвЮ╝ВАїВіхвІѕвІц.
ьі╣ьъѕ В┤ѕЖИ░ В╗┤ьЊеьё░ВЌљВёа, 6bit character codeвЦ╝ ВѓгВџЕьЋўвіћ ВІюВіцьЁюВЮ┤ ВъѕВЌѕВіхвІѕвІц.

ьЉюВцђВЮ┤вљю 1 byte = 8 bits

Ж│аВаЋвљю bitВѕўвАю вгИВъљвЦ╝ ВЮИВйћвћЕьЋўвіћ ASCII(American Standard Code for Information Interchange)Ж░ђ ьЉюВцђВю╝вАю ВъљвдгВъАВъљ, ВъљВЌ░ВіцвЪйЖ▓ї Рђў1 byte = 8 bitsРђЎвАю ВъљвдгВъАЖ▓ї вљўВЌѕВіхвІѕвІц.
ASCIIвіћ ВЏљвъў 7bitвАю 128Ж░юВЮў вгИВъљвЦ╝ ьЉюьўёьЋўВўђВіхвІѕвІц. ВЌгЖИ░ВЌљвіћ ВўЂВќ┤ВЎђ вфЄЖ░ђВДђ ЖИ░ьўИЖ░ђ ьЈгьЋевљўВќ┤ ВъѕВЌѕВіхвІѕвІц.
ВЮ┤вЦ╝ 8bitвАю ьЎЋВъЦьЋўВЌг, 256Ж░юВЮў вгИВъљвЦ╝ ьЉюьўёьЋа Вѕў ВъѕЖ▓їьЋўВЌг Рђў1 byte = 8 bitsРђЎвАю ВѓгВџЕьЋўЖ▓ї вљўВЌѕВіхвІѕвІц.

ВѕФВъљ(Number)ВЎђ Byte

Рђў1 byte = 8 bitsРђЎЖ░ђ ьЉюВцђВю╝вАю Въљвдг ВъАВю╝вЕ┤Вёю, ВѕФВъљвЦ╝ ВађВъЦьЋўЖИ░ ВюёьЋю Ж│хЖ░ёвЈё byteвІеВюёвАю ьЉюВІюьЋўЖ▓ї вљўВЌѕВіхвІѕвІц.
ЖИ░в│ИВаЂВЮИ вІеВюёВЮИ byteВЮў bitВѕў 8ВЮё ЖИ░ВцђВю╝вАю,
8-bit, 16-bit, 32-bit, 64-bit, 128-bit ВаЋВѕўЖ░ђ ВъѕВіхвІѕвІц.
Ж░Ђ bitВѕўВЌљ вћ░вЮ╝, ВаЋВѕўЖ░њ ьЉюьўё в▓ћВюёвіћ, ВЋёвъўВЎђ Ж░ЎВіхвІѕвІц.

\[-(2^{n-1})\sim(2^{n-1}-1)\]

ВўѕвЦ╝ вЊцВќ┤, 8-bit ВаЋВѕўВЮў Ж▓йВџ░, $-(2^7)\sim2^{7}-1$ вАю, $-128\sim127$ВЮў в▓ћВюёвЦ╝ Ж░ќВіхвІѕвІц.

ВЌгЖИ░Вёю,

$n-1$вАю bitВѕўВЌљВёю 1ВЮё ВаюВЎИьЋўвіћ ВЮ┤Вюавіћ,
Ж░ђВъЦ ВЎ╝Вфй bitвЦ╝ вХђьўИ(+, - ВќЉВѕўВЮИВДђ ВЮїВѕўВЮИВДђ вѓўьЃювѓ┤віћЖ▓Ѓ)вЦ╝ ьЉюЖИ░ьЋўвіћвЇ░ ВѓгВџЕьЋўЖИ░ вЋївгИВъЁвІѕвІц.
ВќЉВѕўВЮў в▓ћВюёвіћ, 0 0000000 ВЌљВёю 0 1111111 Ж╣їВДђвАю 0ВЌљВёю 127Ж╣їВДђ Ж░ђвдгьѓхвІѕвІц.
ВЮїВѕўВЮў в▓ћВюёвіћ, 1 0000000 ВЌљВёю 1 1111111Ж╣їВДђ, -128ВЌљВёю -1Ж╣їВДђвЦ╝ вѓўьЃђвЃЁвІѕвІц.
ВќЉВѕў в▓ћВюёВЌљВёю -1ВЮё ьЋўЖ▓ї вљўвіћ ВЮ┤Вюавіћ,
7-bitвАю ьЉюьўёьЋа Вѕў Въѕвіћ ВѕФВъљВЮў Ж░ђВДђВѕўвіћ 128Ж░ю(=$2^7$)ВЮ┤ВДђвДї, ВЮ┤віћ Ж░»ВѕўвЦ╝ ВЮўв»ИьЋўЖ│а, 0ВЮё ьЈгьЋеВІюьѓцвЕ┤ ьЉюьўё Ж░ђвіЦьЋю Вхювїђ ВќЉВѕўЖ░њВЮђ 127ВЮ┤ вљЕвІѕвІц.

Weakly typedВќИВќ┤(javascript)ВЌљВёюВЮў Number

ВЋйьЃђВъЁ ВќИВќ┤(Weakly typed language)віћ в│ђВѕў ВёаВќИВІю в│ёвЈёВЮў typeВЮё ВДђВаЋьЋўВДђ ВЋіВіхвІѕвІц. вЋївгИВЌљ, Вќ┤вќц ВѕФВъљвЦ╝ ВађВъЦьЋўвЇў, ВЮ┤в»И ВаЋьЋ┤ВДё memoryЖ│хЖ░ёВЮё ВѓгВџЕьЋЕвІѕвІц.
javascriptвЦ╝ ВўѕвАю вЊцвЕ┤, numberвіћ вІцвЦИ ьћёвАюЖиИвъўв░Ї ВќИВќ┤ВЮў doubleЖ│╝ Ж░ЎВЮђ Ж│хЖ░ёВЮё В░еВДђьЋЕвІѕвІц.
ВЮ┤ doubleВЮђ ВєїВѕўВаљьЉюьўё(вХёВѕўЖ░њьЉюьўё)ВЮ┤ Ж░ђвіЦьЋЕвІѕвІц.
javascriptВЌљВёа, numberЖ░ђ doubleвАю вІцвБеВќ┤ВДђЖИ░ вЋївгИВЌљ, ВаЋВѕў 32вЦ╝ ВађВъЦьЋўвЇћвЮ╝вЈё, вѓ┤вХђВаЂВю╝вАю ВєїВѕўВаљВЮё ьЈгьЋеьЋю Ж░њВю╝вАю ВађВъЦьЋЕвІѕвІц.
