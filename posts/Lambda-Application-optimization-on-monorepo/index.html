Lambda Application optimization on Monorepo

Monorepo(Monolithic repository)лһҖ?

н•ҳлӮҳмқҳ repository(git repository)м—җм„ң м—¬лҹ¬к°ңмқҳ projectлҘј кҙҖлҰ¬н•ҳлҠ”, repositoryлҘј мӮ¬мҡ©н•ҳлҠ” м „лһөмӨ‘м—җ н•ҳлӮҳмһ…лӢҲлӢӨ.
MSA(MicroService Architecture)м—җм„ң к°Ғ лҸ„л©”мқё лӢЁмң„лЎң м„ңл№„мҠӨлҘј л¶„лҰ¬н•ҳл©ҙм„ң, мқҙ мҪ”л“ңл“Өмқ„ кҙҖлҰ¬н•ҳкё°мң„н•ң м „лһөмңјлЎңм„ң л§Һмқҙ м“°мһ…лӢҲлӢӨ.

BundlingмқҙлһҖ?

Applicationмқ„ кө¬лҸҷн•ҳлҠ”лҚ° н•„мҡ”н•ң мҪ”л“ңл“Өл§Ң лӘЁм•„, packagingн•ҳлҠ”кІғмқ„ л§җн•©лӢҲлӢӨ.
codeлҘј мһ‘м„ұн•ҳл©ҙм„ң, мӢӨм ңлЎң мӮ¬мҡ©н•  мҪ”л“ңмҷҖ к·ёл Үм§Җ м•ҠмқҖмҪ”л“ңлҘј л¶„лҰ¬н•ҳм—¬ кҙҖлҰ¬н•ҳлҠ”кІғмқҖ м–ҙл өмҠөлӢҲлӢӨ. л•Ңл¬ём—җ, Code мһ‘м„ұмӢңм җмқҙ м•„лӢҢ, лі„лҸ„мқҳ лІҲл“Өл§Ғкіјм •мқ„ нҶөн•ҙ, мһ‘лҸҷм—җ н•„мҡ”н•ң мҪ”л“ңл“Өл§Ң filteringн•ҳм—¬ м„ңл№„мҠӨн•©лӢҲлӢӨ.

MonorepoмҷҖ Bundlingмқҙ л¬ҙмҠЁ мғҒкҙҖмқёлҚ°?

MonorepoлҘј мҡҙмҳҒн•ҳлӢӨліҙл©ҙ, к°Ғ serviceм—җм„ң кіөнҶөмңјлЎң мӮ¬мҡ©лҗҳлҠ” мҪ”л“ңл“Өмқҙ мғқк№ҒлӢҲлӢӨ(ex: DBм»Өл„Ҙм…ҳ мғқм„ұмҪ”л“ң). л•Ңл¬ём—җ, мқҙлҹ° мҪ”л“ңл“Өмқ„ кҙҖлҰ¬н•ҳкё° мң„н•ң кіөнҶө лӘЁл“Ҳкіөк°„мқ„ л§Ңл“ӨкІҢ лҗҳлҠ”лҚ°, мқҙл•Ң кіөнҶөлӘЁл“Ҳ(мқҙн•ҳ commonлӘЁл“Ҳ)мқ„ м°ёмЎ°н•ҳлҠ” л°©мӢқмқ„ Bundlingкіјм •м—җм„ң мөңм Ғнҷ”(Optimization)н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙкІҢ мҷң н•„мҡ”н•ңлҚ°?

lambdaлҘј мқҙмҡ©н•ҳм—¬ м„ңл№„мҠӨлҘј мҡҙмҳҒн•ңлӢӨл©ҙ, lambdaмқҳ cold start л¬ём ңм—җ л¶Җл”ӘнһҲкІҢ лҗ©лӢҲлӢӨ. мқҙл•Ң app initializationмӢңк°„мқ„ мөңмҶҢнҷ”н•ҳкё° мң„н•ҙ мқҙлҹ° мөңм Ғнҷ”к°Җ н•„мҡ”н•©лӢҲлӢӨ. lambdaм—җ initialize мӢңк°„м—җ мҳҒн–Ҙмқ„ лҜём№ҳлҠ” мҡ”мҶҢлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.(Lambda performance optimization)
